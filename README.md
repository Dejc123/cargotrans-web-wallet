# Cargotrans Web Wallet

Frontend web wallet for the Cargotrans ecosystem.

## Overview
This project is a frontend web wallet prototype intended for integration with a
decentralized freight exchange and a utility token.

## Features
- Browser wallet connection (MetaMask compatible)
- Wallet address display
- Web3-ready frontend architecture
- Prepared for future utility token integration

## Tech Stack
- HTML
- CSS
- JavaScript

## Security
This project does NOT store private keys, seed phrases, or sensitive user data.
All wallet interactions are handled by the user's browser wallet.

## Project Status
Prototype / early-stage development
